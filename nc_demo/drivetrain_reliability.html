Axle and Hub Upgrades

Hub failures are a known issue with NC Miatas in endurance racing conditions. Upgrading these components is essential for reliability.

Front Hub Upgrades

Upgrade Description Price Range Reliability Improvement
RX8 Front Hubs Direct bolt-on upgrade with higher load rating $150-200 each ★★★★★
Extended Wheel Studs ARP extended wheel studs $100-150 ★★★★☆
Front Wheel Bearings High-quality OEM or Timken bearings $80-120 each ★★★★☆

RX8 vs. MX-5 Front Hub Specifications:

  • MX-5: Dynamic load rating - 33,500N (3,400kgf), Static load rating - 26,800N (2,730kgf)
  • RX-8: Dynamic load rating - 38,500N (3,950kgf), Static load rating - 31,000N (3,200kgf)

Rear Hub Upgrades

Upgrade Description Price Range Reliability Improvement
RX8 Rear Hub Conversion Complete conversion with RX8 uprights, bearings, and hubs $1,200-1,800 ★★★★★
Miatacage Motorsports Competition Rear Hub Upgraded hub using NC axle in existing upright $800-1,200 ★★★★☆
Miata Hubs Billet Enduro Rear Hubs Custom billet aluminum hubs $900-1,300 ★★★★★

Weight Considerations for RX8 Rear Hub Conversion:

  • RX8 upright w/ bearing & hub: 15.7 lbs
  • MX5 upright w/ bearing & hub: 14.7 lbs
  • MX5 axle: 13.6 lbs
  • Modified axle for RX8 conversion: 14.9 lbs
  • Net weight increase: ~2 lbs per side

Conclusion and Recommendations

For maximum reliability in NC Miata endurance racing, prioritize these upgrades:

  1. Cooling System Improvements - This is the most critical reliability factor
  2. RX8 Hub Upgrades - Front is easy, rear is more involved but essential for serious racing
  3. Transmission Upgrades - At minimum, ensure you're using an NC2/NC3 transmission
  4. Moderate Power Increases - Target ~185hp for reliability
  5. Quality Clutch - Choose based on power level, with preference for sprung hub designs

Following this comprehensive approach to drivetrain reliability will significantly improve your NC Miata's chances of completing 8-hour endurance races with 2-hour driver stints without mechanical failures.
